# Loan Approval Prediction using Random Forest Classification

## Overview

This project predicts whether a loan application will be approved or rejected using a Machine Learning model.

## Objective

To build a classification model that helps financial institutions automate loan approval decisions based on applicant details.

## Dataset

The dataset contains:

* Applicant income
* Loan amount
* CIBIL score
* Assets (residential, commercial, luxury, bank)
* Education and employment details

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Workflow

1. Data Loading and Inspection
2. Data Cleaning (handling spaces, formatting)
3. Encoding categorical variables
4. Train-Test Split
5. Model Training using Random Forest Classifier
6. Prediction and Evaluation

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Precision
* Recall

## Results

The model achieved high accuracy and balanced performance, indicating strong predictive capability for loan approval decisions.

## Key Insights

* Financial indicators like income, CIBIL score, and assets play a major role in loan approval
* Machine learning can help automate decision-making in banking systems
